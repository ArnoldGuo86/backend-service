Method Path Description
GET index.html Login page
POST /user/login Login with email/password, and save token to cookie.
GET /heartbeat Get version
GET /user/{token} Get user by specified token
GET /user/id/{id} Get user by specified id
POST /user Create a new user with specified user properties
PUT /user Update an user with specified user properties
DELETE /user/{id} DELETE an user with specified id
POST /user/logout Erase the login user info and remove user token stored
GET /swagger-ui.html swagger-ui
